# ⭐ 별따오기 이름표 생성기

전남대학교 천문동아리 **별따오기** MT용 조별 이름표 자동 생성 프로그램

---

## 📥 다운로드

→ [최신 버전 다운로드 (Releases)](../../releases/latest)

---

## 🖥️ 사용법

1. `별따오기_이름표생성기.exe` 실행
2. 조별 배경 이미지 추가 (파일명에 `1조`, `2조` 포함 시 자동 인식)
3. 명단 CSV/엑셀 불러오기 (A열: 조번호, B열: 이름)
4. 폰트, 위치, 스타일 조정
5. 이름표 전체 생성 → PNG 또는 PDF(A4 4분할)

---

## 📋 명단 파일 형식

| 조 | 이름 |
|----|------|
| 1조 | 홍길동 |
| 1조 | 김철수 |
| 2조 | 이영희 |
