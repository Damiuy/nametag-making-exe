╚══════════════════════════════════════════════╝

▶ 실행 방법
   별따오기_이름표생성기.exe 더블클릭

▶ 사용 순서
  1. [+ 이미지 추가] → 조별 배경 이미지 선택
     - 파일명에 "1조", "2조" 등이 있으면 자동 매핑
     - 예) 1조배경.png, 2조_design.jpg

  2. [명단 파일 불러오기] → CSV 또는 엑셀 선택
     - A열: 조번호 (예: 1조, 2조 ...)
     - B열: 이름
     - 첫 행은 헤더 (조, 이름) 또는 바로 데이터 가능

  3. 폰트 설정
     - [폰트 파일 선택]: .ttf / .otf 파일 직접 선택
     - 웹폰트 URL 입력 후 [다운로드]: 눈누 등에서 복사한 URL 사용

  4. 미리보기에서 이름 위치를 클릭/드래그로 조정
     - ◀ ▶ 버튼으로 다른 이름 미리보기
     - 글자 색, 외곽선, 그림자 설정

  5. [🖨️ 이름표 전체 생성] → 저장 폴더 선택
     - PNG: 조별 개별 파일 (1조_홍길동.png)
     - PDF: 전체 이름표 한 파일 (이름표_전체.pdf)

▶ 명단 CSV 형식 예시
   조,이름
   1조,김예담
   1조,홍길동
   2조,이영희
   2조,박철수
